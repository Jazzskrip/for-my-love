# Анализ `advent-calendar.html`

## Назначение
Одностраничный интерактивный «адвент‑календарь» на 13 дней. Каждый день — карточка с заданием/призом. Доступ к странице закрыт логином и паролем.

## Визуальный стиль и цвета
- Фон страницы: градиент `#0b1026` → `#0f1b3d` → `#0a1a1f` под углом 135°. Поверх — три радиальных световых пятна (золотистые и голубые) с низкой прозрачностью.
- Основной текст: `#e8e0f0` (приглушенный светлый).
- Акцентные золото‑тёплые: `#f6e7b2`, `#f4d06f` (заголовки, номера, бейджи «Купон»).
- Доп. акценты: розовый `#f472b6` (сердечко и конфетти), голубой `#93c5fd` и желтый `#facc15` (конфетти).
- Полупрозрачные стеклянные карточки: фон из тёмно‑синих/зелёных RGBA + `backdrop-filter: blur(10px)`.

## Типографика
- Заголовки и декоративные элементы: `Cormorant Garamond` (Google Fonts), жирность 600.
- Основной текст и карточки: `Nunito` (Google Fonts), 300/400/600.
- Заголовок `h1` выполнен градиентным текстом через `background-clip: text`.

## Компоновка
- Контейнер по центру, max‑width 1400px.
- Сетка карточек: `grid` с `minmax(180px, 1fr)`, gap 20px.
- Карточки квадратные (`aspect-ratio: 1`), с 3D‑перспективой (`perspective: 1000px`).
- Вверху: счетчик прогресса «Открыто X/13», заголовок, подзаголовок.
- Внизу: блок с инструкцией.

## Карточки дней
Лицевая сторона:
- Номер дня (градиентом). Лейбл «День».
- Если день — купон, выводится бейдж «Купон».
- Стиль: стеклянный эффект, золотая рамка, тени.

Обратная сторона:
- Иконка (эмодзи), заголовок и описание награды.
- Маркер «День N» внизу.
- Те же стеклянные эффекты + более выраженное свечение.

Состояния:
- `locked`: полупрозрачная серо‑монохромная карточка, бейдж «ЗАКРЫТО».
- `available`: доступная для открытия.
- `opened`: уже открыта, перевёрнута.

## Эффекты и анимации
- Flip‑анимация: `rotateY(180deg)` с длительностью 0.6s.
- Hover‑подсветка: усиление тени и рамки.
- «Shine» у следующей карточки: диагональный световой пробег (`shineSweep`, 1s).
- Конфетти: канвас поверх всего, частицы летят 1.2s при открытии купонного дня.
- Сердечко (в тексте инструкции): пульсация `pulse` 1.5s.
- В `@keyframes shimmer` для заголовка пустое содержимое — фактически shimmer не работает, хотя анимация задана.

## Открытие дней и логика прогресса
- Дни открываются строго по очереди: доступен только следующий после последнего открытого.
- При клике/тапе доступного дня карточка переворачивается.
- Для купонных дней сначала открывается модальное окно с купоном; подтверждение — кликом по карточке купона.
- После открытия дня прогресс увеличивается и сохраняется в `localStorage`.

## Авторизация
- При загрузке `<body>` имеет класс `auth-locked`: весь контент блюрится и не интерактивен.
- Поверх показана форма логина.
- Данные берутся из `auth.json` через `fetch` (без кеша).
- Проверка — строгое сравнение строк логина/пароля.
- Ошибки показываются в тексте формы.
- Если `auth.json` не загрузился: сообщение «Не удалось загрузить auth.json. Откройте страницу через локальный сервер.»

### Данные для входа
Файл `auth.json` хранит логин/пароль в открытом виде:
- `login`: `123`
- `password`: `1221`

## Хранение прогресса
- Используется `localStorage`.
- Ключ: `adventLastOpened`.
- Значение: номер последнего открытого дня (число).

## Тексты и контент
- Заголовок страницы: «Адвент-календарь для любимой» (в `<title>`).
- В хедере на странице: «Адвент-календарь», подзаголовок «13 дней общения для вас».
- Инструкция: «Нажимай на день, чтобы открыть задание. Дни открываются по очереди.»
- Купонный заголовок: «Специальный Купон!! ??» (двойной символ вопроса в исходнике).

## Важные технические детали
- Страница оптимизирована под iOS: `-webkit-fill-available`, safe‑area inset, фиксы для Safari.
- Отдельные медиазапросы для планшетов, телефонов, очень маленьких экранов и ландшафта.
- `apple-touch-icon` — data‑URI SVG с текстом «??».
- Зависимость от внешних шрифтов Google Fonts (нужен доступ к сети для их подгрузки).
