# Мини-игры

## Описание
На некоторых ячейках нужно пройти мини-игру, чтобы получить награду. Игра появляется в модалке после клика (и после tryAgain, если он есть).

## Конфиг (rewards.json)
```json
{
  "slot": 7,
  "icon": "🎮",
  "title": "Выиграй подарок",
  "desc": "Пройди игру!",
  "game": "flappy"
}
```

Поле `game` — строка с ID игры. Нет поля = нет игры.

## Список игр

| ID | Игра | Сложность | Описание |
|---|---|---|---|
| `coin` | Монетка | Простая | Орёл/решка, угадай сторону |
| `guess` | Угадай число | Простая | Выше/ниже, 5 попыток, захардкоженные числа |
| `roulette` | Рулетка | Средняя | Крутишь колесо, попади в нужный сектор |
| `flappy` | Flappy Bird | Средняя | Canvas, пролети через N труб |
| `invaders` | Space Invaders | Средняя+ | Canvas, стреляй по рядам инопланетян |

## Поток
1. Клик на ячейку (после tryAgain, если есть)
2. Модалка с игрой открывается
3. Выиграл → модалка закрывается, ячейка открывается, награда показывается
4. Проиграл → кнопка "Попробовать снова" внутри модалки

## Файлы
- `js/games.js` — логика всех игр
- `css/games.css` — стили модалок игр
- Каждая игра — отдельная функция, вызываемая по ID
