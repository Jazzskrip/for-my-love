# Анимации открытия ячеек

## Описание
Каждая ячейка может иметь свою анимацию открытия. Стиль: Adult Swim — глитчи, сюрреализм, неожиданные эффекты.

## Конфиг (rewards.json)
```json
{
  "slot": 3,
  "animation": "glitch"
}
```

Поле `animation` — строка с ID. Нет поля = `"flip"` по умолчанию.

## Список анимаций

| # | ID | Эффект |
|---|---|---|
| 1 | `flip` | Переворот карточки (по умолчанию) |
| 2 | `glitch` | Глитч-дёрганье, RGB-смещение, потом открытие |
| 3 | `melt` | Карточка тает/стекает вниз |
| 4 | `shatter` | Разлетается на осколки |
| 5 | `static` | ТВ-помехи на секунду, потом контент |
| 6 | `vortex` | Закручивается в воронку |
| 7 | `dissolve` | Рассыпается на пиксели |
| 8 | `fold` | Складывается как бумага |
| 9 | `rubber` | Растягивается как резина и щёлкает обратно |
| 10 | `blackout` | Экран гаснет, потом карточка появляется |

## Файлы
- CSS-анимации в `css/style.css`
- JS-триггеры в `js/calendar.js`
- Canvas-эффекты (shatter, dissolve) в `js/effects.js`
