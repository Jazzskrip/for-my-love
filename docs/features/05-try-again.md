# Ячейки-обманки (Try Again)

## Описание
Некоторые ячейки не открываются с первого раза. При нажатии показывается рандомная фраза. Открывается только после N попыток.

## Конфиг (rewards.json)
```json
{
  "slot": 10,
  "icon": "🔮",
  "title": "Сюрприз",
  "desc": "Не сразу откроется",
  "tryAgain": { "attempts": 3 }
}
```

Поле `tryAgain` — объект. Нет поля = открывается сразу.

## Пул фраз (data/phrases.json)
```json
{
  "items": [
    "Не-а, попробуй ещё раз!",
    "Почти получилось...",
    "Сегодня не твой день",
    "...всего ~40 фраз"
  ]
}
```

## Поток
1. Клик на ячейку
2. Если попытки < attempts → показать рандомную фразу (toast/тряска), счётчик++
3. Фраза не повторяет предыдущую
4. Счётчик попыток хранится в localStorage
5. Когда попытки >= attempts → пропустить, перейти к game/video/reward

## Комбинации
- tryAgain + game: сначала N попыток, потом игра
- tryAgain + video: сначала N попыток, потом видео
- tryAgain + game + video: попытки → игра → видео → награда
