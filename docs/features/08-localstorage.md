# localStorage

## Описание
Все данные прогресса хранятся в localStorage браузера. Нет бэкенда.

## Ключи

### `adventProgress`
Основной прогресс пользователя.

```json
{
  "openedDays": [1, 2, 3, 5],
  "openedCoupons": [
    { "slot": 3, "icon": "⭐", "title": "Купон", "desc": "Описание" }
  ]
}
```

- `openedDays` — массив номеров открытых дней (отсортирован)
- `openedCoupons` — массив полученных купонов с данными из rewards.json

### `adventTryAgain`
Счётчики попыток для ячеек-обманок.

```json
{
  "10": 2,
  "20": 1
}
```

Ключ = номер дня (строка), значение = количество уже сделанных попыток.
Удаляется для дня после успешного открытия (не нужен больше).

### `adventDebugDate`
Подменённая дата из debug-меню. Если null/отсутствует — используется реальная дата.

```json
"2026-06-03"
```

Строка в формате ISO (YYYY-MM-DD). Сбрасывается опцией "Сброс прогресса" в debug-меню.

## Сброс
Debug-меню "Сброс прогресса" удаляет все три ключа:
- `adventProgress`
- `adventTryAgain`
- `adventDebugDate`

## Ограничения
- Если пользователь очистит данные браузера — прогресс потеряется
- localStorage ограничен ~5MB, наши данные займут < 1KB
