# Разблокировка по дате

## Описание
30 ячеек, по 1 в день. Привязка к START_DATE (29 мая 2026).

## Логика
- START_DATE = 2026-05-29
- День 1 доступен 29 мая, день 2 — 30 мая, и т.д.
- Пропущенные дни можно открыть позже (если сегодня день 5, доступны ячейки 1-5)
- До START_DATE — все закрыты
- Дата берётся с устройства пользователя
- Прогресс (какие дни открыты) хранится в localStorage

## Debug-меню
Уже есть стили в CSS (`.debug-menu-wrap`, `.debug-menu-btn`, `.debug-menu-select`).

Опции:
- День 1 — подменить текущую дату на START_DATE
- День 5 — подменить на START_DATE + 4
- День 15 — подменить на START_DATE + 14
- День 30 — подменить на START_DATE + 29
- Все открыты — разблокировать все 30 ячеек
- Сброс прогресса — очистить localStorage

Debug подменяет "сегодняшнюю дату" в JS, не трогает START_DATE.
