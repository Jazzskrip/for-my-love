# Модалки

## Описание
Каждый тип контента показывается в своей модалке. Модалки не комбинируются — показываются последовательно по потоку открытия.

## Типы модалок

### 1. Игровая модалка
- Когда: поле `game` в rewards.json
- Содержимое: canvas/DOM игры, кнопка "Попробовать снова" при проигрыше
- Закрытие: только после победы (нельзя закрыть оверлеем/Escape)

### 2. Видео-модалка
- Когда: поле `video` в rewards.json
- Содержимое: YouTube iframe (autoplay), таймер обратного отсчёта
- Кнопка "Продолжить" появляется через `buttonDelay` секунд
- Закрытие: только по кнопке (нельзя закрыть оверлеем/Escape пока таймер идёт)

### 3. Купон-модалка
- Когда: поле `coupon: true` в rewards.json
- Содержимое: иконка, название купона, описание, конфетти-эффект
- Закрытие: клик по карточке купона или оверлею

### 4. Награда
- Когда: всегда (после всех предыдущих шагов)
- Содержимое: обратная сторона карточки (icon, title, desc)
- Не модалка — показывается на самой карточке после анимации

## Порядок показа
```
tryAgain (toast, не модалка)
    ↓
Игровая модалка (если game)
    ↓
Анимация открытия карточки
    ↓
Видео-модалка (если video)
    ↓
Купон-модалка (если coupon)
    ↓
Награда на карточке
```

## Общее поведение
- Оверлей: затемнённый фон с blur
- Z-index: модалка выше всего (z-index: 10+)
- Анимация появления: fade-in 0.3s
- На мобильных: модалка занимает почти весь экран (padding: 20px)

Обсуждено. Не реализовано.
