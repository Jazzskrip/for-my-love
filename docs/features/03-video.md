# YouTube видео

## Описание
После открытия ячейки появляется модалка с YouTube видео (автоплей). Кнопка закрытия появляется через N секунд (настраиваемый таймер = длительность видео).

## Конфиг (rewards.json)
```json
{
  "slot": 5,
  "icon": "🎬",
  "title": "Смотри это",
  "desc": "Специально для тебя",
  "video": {
    "url": "https://www.youtube.com/embed/VIDEO_ID",
    "buttonDelay": 10
  }
}
```

Поле `video` — объект. Нет поля = нет видео.

- `url` — embed-ссылка YouTube (формат: `https://www.youtube.com/embed/ID`)
- `buttonDelay` — через сколько секунд появляется кнопка (в секундах)

## Поток
1. Ячейка открывается (после tryAgain/game, если есть)
2. Модалка с iframe: YouTube embed, autoplay=1, rel=0
3. Кнопка скрыта, показывается таймер обратного отсчёта
4. Через buttonDelay секунд — кнопка появляется
5. Клик по кнопке — модалка закрывается, iframe удаляется (видео останавливается)
6. Показывается награда

## iOS ограничение
Автоплей со звуком может блокироваться на iOS. YouTube embed обычно обходит это, но если нет — видео запустится по тапу пользователя.
