# Theme A variables

## Background
- `--bg-gradient`: main background gradient.
- `--bg-spots`: radial glow overlays.

## Typography
- `--text-main`: default text color.
- `--title-gradient`: main title gradient.
- `--subtitle-color`: subtitle text color.

## Progress + user panel
- `--progress-color`: progress text color.
- `--panel-bg`: user panel background.
- `--panel-border`: user panel border.
- `--panel-shadow`: user panel shadow.
- `--panel-name`: user name color.
- `--panel-text`: user panel secondary text.

## Floating info button
- `--fab-border`: info button border.
- `--fab-bg`: info button background.
- `--fab-color`: info button text.
- `--fab-shadow`: info button shadow.

## Info modal
- `--info-overlay-bg`: modal overlay background.
- `--info-card-bg`: modal card background.
- `--info-card-border`: modal card border.
- `--info-card-shadow`: modal card shadow.
- `--info-title`: modal title color.
- `--info-label`: modal term color.
- `--info-row`: modal description color.
- `--info-close-border`: close button border.
- `--info-close-bg`: close button background.
- `--info-close-color`: close button text.

## Cards
- `--card-front-bg`: front face background.
- `--card-front-border`: front face border.
- `--card-front-shadow`: front face shadow.
- `--card-front-hover-border`: hover border.
- `--card-front-hover-shadow`: hover shadow.
- `--card-back-bg`: back face background.
- `--card-back-border`: back face border.
- `--card-back-shadow`: back face shadow.
- `--day-number-gradient`: day number gradient.
- `--day-label-color`: day label text.
- `--reward-title`: reward title text.
- `--reward-desc`: reward description text.
- `--coupon-badge-bg`: coupon badge background.

## Instructions
- `--instructions-bg`: instruction panel background.
- `--instructions-border`: instruction panel border.
- `--instructions-text`: instruction text color.

## Coupon modal
- `--coupon-overlay-bg`: coupon overlay background.
- `--coupon-card-bg`: coupon card background.
- `--coupon-card-border`: coupon card border.
- `--coupon-card-shadow`: coupon card shadow.
- `--coupon-title`: coupon title color.
- `--coupon-text`: coupon text color.
- `--coupon-text-font`: coupon text font.
- `--coupon-title-secondary`: coupon subtitle color.

## Locked state
- `--locked-label`: locked label text.
- `--locked-label-bg`: locked label background.
